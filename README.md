dotfiles
========

Clone this repo and run `install.py` to configure your development environment.

making local customizations
---------------------------

You can make local customizations for some programs by editing these files:
* `vim` : `~/.vimrc_local`
* `zsh` : `~/.zshrc_local`
* `git` : `~/.gitconfig_local`
* `tmux` : `~/.tmux_local.conf`
